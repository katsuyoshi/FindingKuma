# 秋田県河川砂防情報システム 調査結果

## システム概要

秋田県が運用する河川・砂防の防災情報システム。河川水位、雨量、ダム、監視カメラ等のリアルタイムデータを提供している。

- **公式サイト**: https://kasen.pref.akita.lg.jp/
- **スマートフォン版**: https://kasen.pref.akita.lg.jp/sp/html/map.html
- **画像配信サーバー**: https://kasenimg.pref.akita.lg.jp/

## 監視カメラ画像API

### 過去画像（タイムスタンプ指定）

```
https://kasenimg.pref.akita.lg.jp/cameraDataWeb/itv/{YYYYMMDD}/{SN}/{IMAGE_NAME}.jpg
```

| パラメータ | 説明 | 例 |
|-----------|------|-----|
| `YYYYMMDD` | 日付（8桁） | `20260624` |
| `SN` | サイト番号（3桁0埋め） | `050` |
| `IMAGE_NAME` | `image_{SN}_{YYYYMMDDHHmm}` | `image_050_202606241200` |

**完全なURL例**:
```
https://kasenimg.pref.akita.lg.jp/cameraDataWeb/itv/20260624/050/image_050_202606241200.jpg
```

### 最新画像（サムネイル）

```
https://kasenimg.pref.akita.lg.jp/pc/img/camera_site/camera_site_{SN}.jpg
```

- サイト番号は0埋めなし（例: `camera_site_50.jpg`）
- サイズ: 640x480

### 仕様

| 項目 | 値 |
|------|-----|
| 画像形式 | JPEG |
| 画像サイズ | 1280x720（過去画像）/ 640x480（最新サムネイル） |
| 更新間隔 | 5分（:00, :05, :10, :15, :20, :25, :30, :35, :40, :45, :50, :55） |
| 保存期間 | 約1ヶ月（2026年6月25日時点で2026年5月25日以降が取得可能） |
| タイムスタンプ形式 | `YYYYMMDDHHmm`（例: `202606241200`） |

### URLロジック（JavaScriptソースから抽出）

画像URLは `camera_common.js` 内の `getImgSrc` 関数で組み立てられる:

```javascript
function getImgSrc(nw, sn, time) {
  let btime = getCameraTimeValueStr(time);
  let dateStr = btime.substring(0, 8);   // YYYYMMDD
  let imgStr = btime.substring(0, 12);   // YYYYMMDDHHmm

  var result = "https://kasenimg.pref.akita.lg.jp/cameraDataWeb/itv/"
    + dateStr + "/"
    + zeroPadding(sn, 3) + "/"
    + "image_" + zeroPadding(sn, 3) + "_" + imgStr + ".jpg";
  return result;
}
```

- `zeroPadding(sn, 3)`: サイト番号を3桁に0埋め（例: `50` → `050`）
- `getCameraTimeValueStr`: 0時0分の場合に前日24:00に変換する処理あり

### カメラ閲覧ページ

各カメラのスライド表示ページ:
```
https://kasen.pref.akita.lg.jp/static/camera/slide_{SN}.html
```

カメラサイトのグラフ・情報ページ:
```
https://kasen.pref.akita.lg.jp/sp/html/graph.html?dk=15&sn={SN}&nw=1&tm=
```

パラメータ `dk=15` が監視カメラのデータ種別を示す。

## カメラサイト一覧JSON

### エンドポイント

```
https://kasen.pref.akita.lg.jp/pc/data/itv.json
```

- **文字コード**: Shift_JIS
- **形式**: JSON

## 備考

- UIでは「過去24時間のカメラ画像を確認することができます」と表示されるが、サーバー上には約1ヶ月分の画像が保存されている
- 一部のカメラサイト（ID: 3, 4）は `slide_tl_` 形式、（ID: 12, 14, 15, 16, 23, 25, 30, 33, 34, 35, 36, 37, 57, 58）は `slide_g_` 形式のスライドページを使用
- `slide_g_` 形式はグラフ表示（水位グラフとカメラ画像の連動表示）に対応
- カメラ画像は5分毎にキャプチャされるが、0時0分の画像は前日24:00として扱われる特殊処理がある
